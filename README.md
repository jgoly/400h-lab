400m Hurdle Touchdown Calculator

Precision insights for coaches, athletes, and performance analysts.
Transform your 400m hurdle splits into actionable data. Track every touchdown, monitor velocity, and identify sectional strengths and weaknesses in real time.

Why this tool matters:

Instant projections: Updates your projected finish as each split is entered.
Velocity analysis: Visualise speed across each hurdle and compare with target pace.
Coach-ready: Track multiple runs, evaluate performance, and refine strategies.
Effortless setup: Single HTML file, no installation required.

Features
Calculator Tab
Enter cumulative touchdown times for H1–H10 and the finish.
View hurdle intervals, versus-target deltas (green/red), and projected 100m, 200m, 300m split checkpoints instantly.
Target splits panel shows the precise times required at each hurdle to reach your goal.
Projected finish updates dynamically from H2 onwards.
Partial run support: enter as few or as many hurdles as you have data for.
Velocity profile chart compares actual versus target pace for each segment.
100m sectional breakdown: 0–100, 100–200, 200–300, 300–finish, plus first vs second 200m and differential.
Save up to 20 runs in-browser and reload at any time.
Optional stride pattern input per hurdle for advanced analysis.
Pace Guide Tab
Input any target time and receive a complete split-by-split breakdown.
100m, 200m, 300m checkpoints clearly highlighted.
Interval change column shows expected slowdowns (red) or pace maintenance (green).
Target velocity profile for visual comparison.

Methodology
Developed from biomechanical analysis of elite competitions:
IAAF World Championships Berlin 2009
London Diamond League 2017
2025 World Championships, Tokyo
Hurdle positions: H1 = 45m, H2–H10 spaced 35m apart, finish = 400m.
100m checkpoints: Interpolated using segment velocity for precision.
Projection model: Uses a quadratic formula to predict finish from actual splits; updates dynamically with input. Target changes do not affect projections.
H1 note: H1 projection is suppressed due to high variability in block reaction and acceleration phase.
